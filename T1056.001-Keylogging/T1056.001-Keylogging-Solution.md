# Solution Walkthrough: T1056.001 (Keylogging)

### Step 1: Reconnaissance
Rogue administrators or previous attackers often leave keylogger output files in temporary or hidden directories. Search common temp directories (`/tmp`, `/var/tmp`, `/dev/shm`) for hidden folders or files.
`ls -la /var/tmp/`
You will discover a hidden directory named `.hidden`.

### Step 2: Extraction
List the contents of the hidden directory.
`ls -la /var/tmp/.hidden/`
You will see a file named `klog_out.txt`.

### Step 3: Analysis
Read the file to analyze the captured keystrokes.
`cat /var/tmp/.hidden/klog_out.txt`
You will observe a recorded SSH session, capturing the username, target IP, the plaintext password typed by the victim, and the challenge flag.
